1. Viewport Meta Tag

The foundation of responsive design - telling the browser how to control the page's dimensions and scaling.

Essential Viewport Meta Tag

<meta name="viewport" content="width=device-width, initial-scale=1.0">

This page uses the viewport meta tag to ensure proper scaling on mobile devices.

Content scales properly

2. Media Queries

Apply different styles based on device characteristics like screen size, orientation, and resolution.

Basic Media Query Syntax

/* Mobile First Approach */
.container {
    padding: 1rem;
}

/* Tablet and up */
@media (min-width: 768px) {
    .container {
        padding: 2rem;
    }
}

/* Desktop and up */
@media (min-width: 1024px) {
    .container {
        padding: 3rem;
    }
}

Responsive Box

This box changes size and color based on screen width!

Common Breakpoints

/* Mobile: 320px - 767px */
@media (max-width: 767px) { }

/* Tablet: 768px - 1023px */
@media (min-width: 768px) and (max-width: 1023px) { }

/* Desktop: 1024px and up */
@media (min-width: 1024px) { }

/* Large Desktop: 1200px and up */
@media (min-width: 1200px) { }

3. Fluid Typography

Create text that scales smoothly across all device sizes using relative units and CSS functions.

Using clamp() for Fluid Typography

h1 {
    font-size: clamp(1.5rem, 4vw, 3rem);
    /* Minimum: 1.5rem, Preferred: 4vw, Maximum: 3rem */
}

p {
    font-size: clamp(1rem, 2.5vw, 1.25rem);
    line-height: 1.6;
}

Fluid Typography

This text scales smoothly between different screen sizes. Resize your browser window to see the effect!

Responsive Line Heights

p {
    font-size: clamp(1rem, 2.5vw, 1.25rem);
    line-height: clamp(1.4, 1.2 + 0.5vw, 1.8);
    /* Responsive line height that adapts to font size */
}

This paragraph demonstrates responsive line height. The spacing between lines adjusts automatically as the text size changes, ensuring optimal readability on all devices.

4. Flexible Images

Make images responsive and ensure they look great on all devices.

Basic Responsive Images

img {
    max-width: 100%;
    height: auto;
    display: block;
}

/* For different aspect ratios */
.responsive-image {
    width: 100%;
    height: 0;
    padding-bottom: 56.25%; /* 16:9 aspect ratio */
    background-image: url('image.jpg');
    background-size: cover;
    background-position: center;
}

5. Mobile-First Design

Start with mobile design and progressively enhance for larger screens.

Mobile-First Approach

/* Mobile styles (base) */
.navigation {
    display: none; /* Hidden by default */
}

.mobile-menu {
    display: block;
}

/* Tablet and up */
@media (min-width: 768px) {
    .navigation {
        display: flex;
    }
    
    .mobile-menu {
        display: none;
    }
}

6. Real-world Examples

See responsive design principles in action with complete layouts.

Responsive Card Layout

.card-grid {
